Low Poly Rigged Animals




Folders:

1) Material : Contain all different material for the models (To create new material copy an existing asset, press ctrl+D and change the color ) 
   
2) Models : Contain all the models in .FBX format   (Change the model with your 3d editor if you want to modify something)

3) Prefabs : Contain all the prefabs to use in your scene

4) Scenes : Contain 2 different scenes 

5) Script : Contain a flyng camera script to move around in the scene.

6) Texture: Contain all textures for the models
